
MODEL COMPARISON REPORT — DRAFT

Classical model:
The primary classical model is XGBoost trained on lagged macroeconomic, shipping, policy, local FAO, and regional inflation features. It uses recursive 12-month forecasting to simulate the real challenge setup.

Deep learning model:
The deep learning benchmark is a small two-layer LSTM with 32 hidden units, dropout=0.30, AdamW optimisation, weight decay, gradient clipping, and early stopping. The model uses a 24-month lookback window and directly predicts a 12-month horizon.

Validation:
The classical model is tested using 12-month forward validation windows. The LSTM is validated using a direct 2022 forecast from history ending December 2021. Metrics are saved in model_comparison_metrics.csv.

Winner:
Validation winner: Classical_XGBoost_or_GBR with average RMSE=4.392.

Analytical conclusion:
The comparison is intentionally honest. With roughly 288 monthly observations, the classical model is expected to be competitive because it can exploit structured lag features without needing large sample sizes. The LSTM is included to satisfy the Deep Learning requirement and to demonstrate whether sequence learning adds value under small-data constraints.
